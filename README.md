# Predictive Maintenance with Machine Learning

Este trabajo consiste en encontrar un modelo de aprendizaje automático adecuado para el mantenimiento predictivo.

El conjunto de datos utilizado fue creado mediante un script y corresponde a un conjunto de datos sintético de turbinas de vapor.

En este trabajo, primero se realizó un análisis exploratorio de datos (EDA) y, posteriormente, se seleccionó el modelo más adecuado y se evaluó su eficiencia general.
